# InfinityZoomWeb

Public privacy policy website for Infinity Zoom:

- Google Play: `com.sdlckdrl.infinitezoom`
- Galaxy Store: `com.sdlckdrl.infinitezoom.galaxy`
- Published page: <https://sdlckdrl.github.io/InfinityZoomWeb/privacy-policy.html>

AdMob 출시 준비와 `app-ads.txt`의 실제 배포 위치는 [AD_MOB_RELEASE_SETUP.md](AD_MOB_RELEASE_SETUP.md)를 따른다. 이 저장소의 [app-ads.txt](app-ads.txt)는 원본 파일이며, AdMob 인증에는 도메인 최상단의 `/app-ads.txt`로 별도 배포해야 한다.
